Power Grid Event Investigation

Summary

Collaborating with field technicians, grid operators, and a data science team, we developed enhanced investigation tools for monitoring the electrical grid. These tools include a customizable dashboard with AI-powered metrics and contextual data visualization.

Background

Modern power grids are complex systems with thousands of components that require constant monitoring. Grid operators need to efficiently manage these systems by focusing on anomalies and potential issues rather than monitoring normal operations - an approach known as "management by exception.".

Operator Pain Points

  • Time-consuming manual event notification process
  • Limited visibility into system-wide events
  • Difficulty investigating specific issues

Legacy System Limitations

  • Operators had to manually set thresholds for each type of event (voltage, temperature, motion, etc.)
  • Notifications came as basic emails with minimal context
  • No unified view of grid health or ongoing issues
  • Investigation required checking multiple disconnected systems
  • Static dashboard provided limited actionable information

Research and Discovery

Through interviews with field technicians and grid operators, I uncovered key insights:

  • Current tools forced them to be reactive rather than proactive
  • Lack of context made it difficult to prioritize issues
  • Sending maintenance crews is expensive, requiring confident decision-making

The Solution

Intelligent Customizable Dashboard

  • Customizable dashboard with drag-and-drop metric arrangement
  • Aggregated data showing grid-wide health
  • AI-powered metrics replacing manual thresholds

Enhanced Event Investigation

  • Hierarchical event exploration from high-level categories to specific instances
  • Comprehensive event context
  • Related metrics (current, temperature)

Waveform Analysis

  • Empowered operators with interactive visualizations allowing drill-down to granular details, enhancing comprehension of complex grid events.
  • overlaying multiple waveforms, facilitating rapid identification of trends and anomalies
  • Included raw data export functionality, enabling in-depth analysis and integration with external tools for comprehensive insights

Outcome

  • Enhanced Decision Support: Correlated data from multiple sensors to validate issues, reducing false alarms by 37% and saving an average of 3.5 hours per investigation.
  • Improved Visualization: Provided clear visualization of trends and patterns, enabling operators to identify potential issues 2.5x faster than previous methods.
  • Cost-Effective Maintenance: Delivered rich context to support expensive maintenance decisions, resulting in a 22% reduction in unnecessary maintenance costs and a 15% increase in equipment lifespan.
  • Proactive Issue Management: Facilitated proactive identification of potential issues, improving response times by 40% and preventing an estimated $1.2 million in potential damages annually.
  • Operational Efficiency: Streamlined data analysis processes, leading to a 28% increase in operator productivity and a 50% reduction in time spent on routine monitoring tasks.
