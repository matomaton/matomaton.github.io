Dashboard Customization

Design Challenge

A key challenge in this dashboard design was keeping elements organized while letting users customize it. As users resize or move elements, the layout maintains a natural left-to-right, top-to-bottom flow - similar to how we read text.

Communicating the Interaction Vision

I created different artifacts to show exactly how I wanted elements to move and reorder. This helped engineers check if it was technically possible, while giving stakeholders and the product team a clear preview of the user experience.

Element Expansion

When users expand an element, nearby elements automatically adjust their positions. This keeps everything flowing naturally from left to right and top to bottom, without creating awkward spaces or breaking the layout's order.

Element Arrangement

When users drag an element to a new position - like moving one from the bottom right to the middle - other elements shift out of the way. Everything stays in order, flowing from left to right and top to bottom, making it easy for users to predict how the layout will change.
