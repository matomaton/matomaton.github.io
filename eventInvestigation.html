Power Grid Event Investigation

Summary

Collaborating with field technicians, grid operators, and data scientists, we developed investigation tools for monitoring the electrical grid. These tools include a customizable dashboard, AI-powered metrics, and contextual data visualizations.

Background

Modern power grids are complex systems with thousands of components that require constant monitoring. Grid operators need to efficiently manage these systems by focusing on anomalies and potential issues rather than monitoring normal operations - an approach known as "management by exception.

Operator Pain Points

  • Cumbersome manual event notifications setup
  • Limited visibility into system-wide events
  • Difficulty investigating specific issues

Legacy System Limitations

  • Operators had to manually set thresholds for each type of event (voltage, temperature, motion, etc.)
  • Notifications came as basic emails with minimal context
  • No unified view of grid health or ongoing issues
  • Investigation required checking multiple disconnected systems
  • Static dashboard provided limited actionable information

Research and Discovery

I uncovered insights through interviews with field technicians and grid operators.

  • Current tools force them to be reactive rather than proactive
  • Lack of context makes it difficult to prioritize issues
  • Sending crews to fix equipment is expensive

Data Science Collaboration

I partnered with the data science team

  • Defined key metrics for different types of grid events
  • Identified machine learning abilites for predictive notifications
  • Discoved opportunites for automated anomaly detection

The Solution

Intelligent Customizable Dashboard

  • Drag-and-drop prioritized arrangement of metrics
  • Aggregated data showing grid-wide health
  • AI-powered metrics replacing manual thresholds

Enhanced Event Investigation

  • Ability to explore high-level categories
  • Filters designed to isolate specific events and reveal detailed insights
  • Contextal metrics (current, temperature) telling a more complete story

Waveform Analysis

  • Enabled operators to drill down into granular details with interactive visualizations, clarifying complex grid events
  • Overlaying waveforms made it easier to spot trends and detect anomalies
  • Offered data export capabilities to support in-depth analysis and integration with other tools

Outcome

  • Enhanced Decision Support: Aggregated sensor data validated issues and improved notification accuracy, reducing noise by 37%
  • Improved Visualization: Trend and pattern visualizations allowed operators to identify potential issues nearly twice as fast
  • Proactive Issue Management: Faster response times and earlier insights enabled proactive maintenanc
  • Cost-Effective Maintenance: The improvements fostered confident decisions, lowering maintenance expenses and increasing equipment lifespan
